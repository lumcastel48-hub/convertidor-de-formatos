## 📌 Descripción

**MultiConvert Pro** es una aplicación web HTML5 que permite convertir múltiples formatos de documentos de forma rápida, intuitiva y sin necesidad de servidores.

Diseñada como un ecosistema de conversión todo-en-uno, facilita la transformación de contenido entre formatos populares como Markdown, PDF, Word, HTML y Excel.

---

## ✨ Características principales

- 🔄 Conversión multi-formato en una sola aplicación
- ⚡ Procesamiento 100% en el navegador (sin backend)
- 👀 Vista previa en tiempo real
- 📂 Soporte para entrada de texto y archivos (drag & drop)
- 📥 Descarga automática de resultados
- 🎨 Interfaz moderna y responsiva

---

## 🔁 Formatos soportados

### 📘 Markdown
- MD → HTML
- MD → PDF
- MD → DOCX

### 📄 Documentos
- DOCX → PDF
- DOCX → HTML
- TXT → PDF
- PDF → DOCX *(limitado)*

### 🌐 Web & Datos
- HTML → PDF
- HTML → DOCX
- Excel (XLSX) → PDF
- Excel (XLSX) → HTML
